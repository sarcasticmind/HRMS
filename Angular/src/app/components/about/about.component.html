ABOUT

This is a simple website for human resources managment system (HRMS) where an HR can add new employees and edit a current one and delete the employee if he wants to.
Also he/she can submit a new record for the employee attendence or edit an existing one.
The HR has the full freedom to set the official holidays for the year and set the weekly vacations too.
Finally using the former details the HR can calculate and print the monthly salary report for the employees of the company. employee
